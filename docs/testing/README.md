Этот раздел собирает в одном месте методику и результаты тестирования релизной версии расширения.

## Состав раздела
- `datasets.md` — источники тестовых данных и причины их выбора.
- `unit-tests.md` — схема проведения unit-тестов и их результаты.
- `integration-tests.md` — методика интеграционных запусков на release-сборке.
- `user-testing.md` — пользовательское тестирование и профиль основной выборки.
- `summary.json` — структурированная сводка ключевых метрик.

## Как запускать
```bash
npm test
npm run test:unit
npm run test:integration
npm run test:coverage
```

Техническая структура:
- `tests/unit/*.test.mjs` — unit-тесты по модулям и core-логике.
- `tests/integration/run-integration.mjs` — integration runner.
- `tests/integration/scenarios.mjs` — генератор сценариев на `348` прогонов.
- `tests/helpers/chrome-mock.mjs` — mock `chrome` для unit-тестов data layer.

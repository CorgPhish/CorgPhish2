# Security Policy

## Supported Versions

Поддерживаются последние релизные версии расширения.

## Reporting a Vulnerability

Если вы нашли уязвимость:
1. Не публикуйте детали в открытом issue до исправления.
2. Создайте приватное сообщение владельцу репозитория.
3. Приложите:
- шаги воспроизведения
- версию расширения и браузера
- ожидаемое и фактическое поведение
- при наличии: скриншоты и логи

После подтверждения будет подготовлен патч и релиз с исправлением.

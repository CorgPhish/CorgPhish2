# Руководство пользователя CorgPhish

## Что делает расширение

CorgPhish проверяет сайты прямо в браузере и помогает остановить:

- фишинговые страницы;
- поддельные домены, похожие на известные сайты;
- подозрительные формы ввода;
- опасные скачивания;
- переходы по рискованным ссылкам.

Расширение работает локально в браузере и не требует внешнего API для базовой проверки.

## Как установить

1. Откройте `chrome://extensions`.
2. Включите `Developer mode`.
3. Нажмите `Load unpacked`.
4. Выберите папку `apps/extension/`.
5. После установки закрепите расширение в панели браузера.

## Что показывает popup

В popup доступны основные состояния:

- `Сайт безопасен` — домен совпал с trusted-списком или белым списком.
- `Требует проверки` — сайт не подтверждён, но явной жёсткой блокировки нет.
- `Опасный сайт` — найдены признаки фишинга.
- `Сайт заблокирован` — домен находится в чёрном списке.

Также из popup можно:

- перепроверить текущую вкладку;
- проверить все открытые вкладки;
- открыть историю проверок;
- открыть настройки;
- добавить сайт в белый или чёрный список;
- отправить жалобу на фишинговый сайт.

## Что происходит при опасном сайте

Если сайт признан опасным, CorgPhish может:

- заблокировать переход на страницу;
- заблокировать отправку формы;
- заблокировать скачивание файла;
- открыть отдельный экран блокировки с безопасными действиями.

На экране блокировки доступны кнопки:

- `Назад` — вернуться на предыдущую страницу;
- `Открыть` — временно разрешить сайт на 5 минут;
- `Добавить в чёрный список` — блокировать сайт в дальнейшем;
- `Сообщить о фишинге` — открыть форму жалобы.

Если сайт маскируется под доверенный домен, может также появиться кнопка перехода на официальный сайт.

## Основные настройки

В настройках доступны:

- язык интерфейса;
- тема;
- автопроверка при открытии страницы;
- подсветка подозрительных ссылок;
- строгий режим;
- блокировка ввода и загрузок;
- срок хранения истории;
- компактный режим.

## Белый и чёрный списки

### Белый список

Используйте белый список для сайтов, которым вы доверяете и хотите считать безопасными вручную.

### Чёрный список

Используйте чёрный список для сайтов, которые нужно блокировать всегда.

## Как правильно пользоваться

- Проверяйте адрес сайта перед вводом логина, пароля и кода.
- Не отключайте блокировку ввода без необходимости.
- Если сайт выглядит подозрительно, лучше закрыть его, а не разрешать вручную.
- Для спорных случаев сначала откройте историю и перепроверьте сайт.
- Если видите явный фишинг, отправьте жалобу через встроенную кнопку.

## Что делать, если расширение не обновило статус

1. Нажмите `Перепроверить` в popup.
2. Обновите страницу.
3. Перезагрузите расширение в `chrome://extensions`.
4. Убедитесь, что открыта обычная `http/https` страница, а не системная вкладка браузера.

## Что делать, если сайт заблокирован ошибочно

1. Не вводите данные сразу.
2. Проверьте адрес вручную.
3. Если сайт действительно безопасен, нажмите `Открыть`.
4. Если это постоянный безопасный сайт, добавьте его в белый список.

## Ограничения

- Расширение проверяет только `http` и `https`.
- Системные страницы браузера не анализируются.
- Некоторые сайты с жёсткими сценариями JavaScript могут требовать повторной перепроверки.
- Временное разрешение действует 5 минут и не должно использоваться для неизвестных сайтов.

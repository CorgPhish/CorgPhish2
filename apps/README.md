# Apps

Каталог `apps/` содержит исполняемые продукты репозитория.

Сейчас здесь один продукт:

- `extension/` — Chrome/Chromium расширение CorgPhish

Такая схема оставляет место для будущих приложений, например отдельного demo-сайта или панели администратора, не смешивая их с кодом самого расширения.

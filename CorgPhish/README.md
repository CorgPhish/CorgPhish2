# CorgPhish

Локальное Chrome-расширение (Manifest V3), которое проверяет активную вкладку по списку доверенных доменов и вашему собственному белому списку. ML и контент-скрипты убраны: вся логика теперь работает в попапе на основе локальных данных.

## Возможности
- Проверка домена активной вкладки против `trusted.json` + пользовательского whitelist.
- Двухъязычный интерфейс (ru/en), светлая/тёмная темы.
- История последних проверок (до 50 записей) в `chrome.storage.local`.
- Автопроверка при открытии попапа, опциональные предупреждения для подозрительных доменов.
- Вручную проверяйте любой домен (без перехода) и быстро добавляйте его в whitelist из основного экрана.
- Быстрый обзор: счётчики доверенных/подозрительных доменов и размера вашего whitelist.
- Доп. настройки: автоочистка истории (7/30/90 дней или отключено), компактный режим интерфейса.
- Код попапа разбит на ES-модули: `popup/config|data|dom|inspection|i18n|ui|main`.
- Автоблок на подозрительных сайтах: контент-скрипт отключает формы/загрузки и показывает предупреждение с опцией «разблокировать на 5 минут». Системные уведомления при риске (настраиваются).
- Предупреждение срабатывает сразу при открытии подозрительной страницы: загрузки блокируются, формы запрещены, системное уведомление и alert появляются без необходимости открывать попап.

## Установка (unpacked)
1. `chrome://extensions` → включить Developer mode.
2. Нажать **Load unpacked** и выбрать папку `CorgPhish/`.

## Структура
- `manifest.json` — декларация MV3 без фоновых/контентных скриптов.
- `popup.html|css|js` — UI и логика проверки.
- `trusted.json` — локальный каталог доверенных доменов (можно дополнять через UI, хранится в `chrome.storage.local`).

## Использование
- Откройте попап на странице HTTP/HTTPS. Если домен есть в каталоге/whitelist — статус «доверенный», иначе — предупреждение.
- Переключение языка/темы и управление whitelist — в настройках попапа.
- Историю можно очистить с помощью кнопки «Очистить» в разделе History.
- Для адресов из писем или чатов используйте карточку «Проверить домен вручную» — можно вводить домен или полный URL.
- Если домен не доверенный, нажмите «Добавить в белый список» прямо на главной карточке статуса.
